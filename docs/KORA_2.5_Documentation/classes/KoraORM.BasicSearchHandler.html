\KoraORMBasicSearchHandler

A base class for search handlers that deal with Kora basic search.

All search handlers are expected to implement this interface.

printForm()

printForm(array $args)

Prints a basic search form with a text box and a search button.

The request parameters produced by this search form should be able to be handled by the implementation of \KoraORM\SearchHandler::handleRequest() for this handler. The arguments for this method are expected in the form of an associative array.
